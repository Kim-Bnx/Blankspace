### Comment déplacer ou supprimer des éléments d’une page

Commencez par repérer dans quel template se trouve l’élément en vous aidant du `schémas des templates.`

Dans les templates, chaque élément est identifié par des commentaires entre `<!-- ... -->` et vous pouvez déplacer les lignes HTML correspondantes où vous le souhaitez.

Il ne faut pas avoir peur de faire plusieurs essais pour obtenir le résultat voulu !

### Affichage déformée ou impossible d’une page

Si vous avez procédé à une installation manuelle, il y a de forte chance que vos templates soient mal installés. Il y a beaucoup de code à copier/coller dans de nombreuse page, une erreur de manipulation est vite arrivée. Assurez-vous que le contenu de votre template correspond au code de la documentation.

### Les causes principales d’un dysfonctionnement de script

Assurez-vous que :

- l’option `activer la gestion des scripts` est sur `oui` dans la page de gestion du javascript.
- le code javascript soit activé dans l’endroit désiré (option de placement : sur l’index, sur les sujets, sur toutes les pages, …).
- il n’y a aucun message d’erreur relatif à votre problème dans la console du navigateur (F12 → console).
- si vous avez fait des modifications HTML, vous n’auriez pas retirer une `.class` qui pouvait être nécessaire au bon fonctionnement du script.
- s’il s’agit d’un script provenant d’un tutoriel extérieur, qu’il puisse être modifié pour être utilisé sur le Blank. Très souvent il s’agit d’un problème de `.class` qui permettent d’identifier des éléments de la page, comme évoqué dans le point ci-dessus.

### Avatar du dernier inscrit qui ne s’affiche pas

Assurez-vous d’avoir rempli tout les prérequis pour le bon fonctionnement du script :

- activer la gestion des scripts (voir question ci-dessus)
- avoir l’option `activer le profil avancé` sur `non` dans `Utilisateurs & groupes › Profil › Options générales`
- que les nouveaux inscrits aient un avatar par défaut

### Impossible de déplacer les champs de profil

L'orthographe exacte pour que le script des champs de profil comprenne quel champs vous ciblez est la suivante :

- tout en minuscule, exemple : `*'messsage'*`
- les champs en plusieurs mots sont séparés par un tiret, exemple : `*'etat-civil'*`
- pas de caractères spéciaux ou d'accents, exemple : `'credits'`

### Menu déroulant de la version mobile absent

Le menu déroulant de la version mobile s’affichage lorsque l’ID `id="header"` est présent dans le template `overall_header` Par défaut dans le Blank, il est présent dans la balise de la bannière. Assurez-vous de ne pas l’avoir retiré.
